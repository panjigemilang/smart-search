# Smart Search Image Extension

A futuristic Chrome extension that allows you to search for images by screenshotting and cropping any area of your screen.

## Features
- **Capture & Crop**: Precise snipping tool style interface.
- **Search Engines**: Supports Google Images (more coming soon).
- **Design**: Sleek, iOS-style glossy interface with neon accents.
- **Privacy**: Runs locally and uploads only when you confirm.

## Installation
1. Open Chrome and navigate to `chrome://extensions`.
2. Enable **Developer mode** in the top right corner.
3. Click **Load unpacked**.
4. Select this directory (`d:\panji\smart-search`).

## Usage
1. Click the extension icon.
2. Click the large "Scan" button.
3. Your screen will dim. Click and drag to select an area.
4. Click "Search" to find similar images on Google.
